Types of Attacks and Descriptions

Attack Type Description
TCP Attack targeting the Transmission Control Protocol (TCP) by sending repeated connection requests to overwhelm the server or network.
UDP Attack targeting the User Datagram Protocol (UDP) by sending unexpected data packets, leading to server disruption.
TCPM Same thing, but it sends both a message and a packet.
UDPM Same thing, but it sends both a message and a packet.
UCP Attack combining TCP and UDP protocols, using both to increase the size and impact of the attack.
PING PING Flood attack; sending excessive "Ping" requests to the server, leading to server unresponsiveness by overloading its resources.
NTP NTP Amplification attack; exploiting the NTP protocol to send large responses to a targeted system, causing it to be overwhelmed.
HTTPV1 (GET) Attack using the first version of the HTTP protocol with GET requests, flooding the server with numerous requests to overload it.
HTTPV2 (POST) Attack using the second version of the HTTP protocol with POST requests, taking advantage of new features for a more potent attack.
HTTPV3 (Downloads) An attack using the HTTP/3 protocol with POST requests to repeatedly download large files, consuming bandwidth and slowing down the server.
SYN SYN Flood attack; exploiting the TCP protocol by sending false connection requests, consuming server resources without establishing proper connections.
IIPORT Attack ports from 1 to the maximum port number, Causing disruption by targeting multiple ports.
IRPORT Attack random ports on the server, causing disruption by targeting unexpected ports.
ARP ARP Spoofing attack; exploiting the ARP protocol to change MAC addresses on the network, redirecting traffic to the attacker.
DNS DNS Spoofing attack; exploiting vulnerabilities in DNS services to redirect requests to harmful or unauthorized sites.
SMB Attack exploiting vulnerabilities in the SMB (Server Message Block) protocol to access data over the network.
SNMP Attack exploiting the SNMP protocol to gather sensitive information or alter network device configurations.
SMTP SMTP (Simple Mail Transfer Protocol) is a protocol used for sending and relaying email messages between email servers.
MEMCAHED Memcached attack; exploiting Memcached servers over UDP in an Amplification attack to overwhelm the targeted servers.
FTP Attack through the FTP protocol; attempting unauthorized access by using Brute Force to guess passwords.
TELNET Attack through the Telnet protocol; exploiting vulnerabilities in Telnet to gain unauthorized access to servers via scripts.
CHARGEN Chargen attack; exploiting the Chargen protocol to flood the server with useless text data, reducing the server's processing capacity.
LDAP LDAP Injection attack; exploiting vulnerabilities in the LDAP protocol to access unauthorized data or execute harmful queries.
CLDAP CLDAP attack; exploiting CLDAP servers in an Amplification attack, sending small requests that generate large responses directed at the target.
